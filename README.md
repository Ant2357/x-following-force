## 機能(Feature)
X のホームを開いたときに自動で「フォロー中」タブに切り替えます。

## 使い方(Usage)
1. [chrome://extensions/](chrome://extensions/)  を開いてデベロッパーモードを ON にします
2. パッケージ化されていない拡張機能をクリックします
3. 当フォルダを選択します
4. これで導入完了です！

上記の手法で、すぐに導入が完了します。
